# 마크다운 문법 정리

## 설정 방법

1. 제목
- #을 붙여 크기를 조절한다
- ex) # 제목 ## 소제목 ### 소소제목

2. 강조
- **이나 __를 문장 사이에 반복 사용하여 글을 강조한다.
- __는 단어 사이에 사용 할 수 없다. 
- ex) **안녕** 하세요.  안녕 __하세요__.
- *이나 _를 문장 사이에 반복 사용하여 글을 기울인다.
- ex) *홍순혁* _홍길동_

3. 불러오기
- [이름] (주소)를 작성하여 해당 주소로 이동 할 수 있다.
- ex) [네이버](https://www.naver.com/)
